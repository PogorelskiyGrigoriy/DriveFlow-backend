# Когда ученик кликнул по Magic-ссылке, он попадает на главный экран
1. Блок А: Виджет «Ближайший урок». Если у ученика есть урок со статусом SCHEDULED, эта карточка висит на самом верху крупным шрифтом: «Следующее занятие: Завтра, 19 мая в 11:00».
! Рядом — кнопка «Отменить урок» !
2. Блок Б: Кнопка «Записаться на новый урок». Крупная кнопка, которая открывает календарь доступных слотов
3. Блок В: Быстрая связь с учителем. Маленькая плавающая кнопка (или строчка в подвале) с именем его инструктора
 (например, «Твой инструктор: Eli Cohen») и иконкой телефона/WhatsApp

# Экран бронирования (Booking Calendar)
Ученик видит сетку дней (например, на неделю вперед).
Кликает на день -> видит только свободные часы (которые наш бэкенд отфильтровал по графику учителя).
Кликает на час -> всплывает окно подтверждения: «Записаться на завтра в 14:00?» -> Кнопка «Подтвердить».
